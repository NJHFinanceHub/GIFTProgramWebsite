Welcome! This folder contains your complete GIFT website.

┌─────────────────────────────────────────────────────────────────┐
│  STEP 1: OPEN IN VS CODE                                        │
└─────────────────────────────────────────────────────────────────┘

1. Right-click this folder
2. Select "Open with Code"
   (or drag this folder into VS Code)

┌─────────────────────────────────────────────────────────────────┐
│  STEP 2: START A LOCAL SERVER                                  │
└─────────────────────────────────────────────────────────────────┘

EASIEST METHOD (Recommended):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. In VS Code, install "Live Server" extension
2. Right-click index.html → "Open with Live Server"
3. Done! Website opens in your browser automatically.

ALTERNATIVE METHOD (Using Terminal):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Open VS Code Terminal (Terminal → New Terminal), then type:

    python -m http.server 8000

Then open your browser to: http://localhost:8000
